# Infrastructure Docker Artifacts

This directory mirrors the production/development runtime artifacts used by the project root.

## Files

- `Dockerfile`: use root `Dockerfile`
- `docker-compose.dev.yml`: use root `docker-compose.dev.yml`
- `docker-compose.prod.yml`: use root `docker-compose.prod.yml`
- `nginx.conf`: use root `nginx/nginx.conf`

These are documented here to keep module-level infra design (`design/modules/09_*`) aligned with repository structure.
